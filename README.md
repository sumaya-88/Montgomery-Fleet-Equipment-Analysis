# Montgomery-Fleet-Equipment-Analysis

Final project for IBM/Coursera Excel Basics for Data Analysis — data cleaning, preparation, and pivot table analysis of Montgomery County's government fleet equipment inventory.

## 📌 Project Overview

This project simulates the role of a **Junior Data Analyst** working in a local government office. Using real-world fleet inventory data from Montgomery County, the goal was to clean, prepare, and analyze the dataset using **Excel Basics for Data Analysis** course on Coursera.

## 📋 Dataset

- 1,582 equipment records across 12 county departments
- Key columns: Department, Equipment Class, Equipment Count
- Source: Montgomery County Open Data Portal (Public Domain)

## 🛠 Tools Used

- Microsoft Excel for the Web
- Pivot Tables
- Filter, Remove Duplicates, Find & Replace, Flash Fill, Conditional Formatting

## 📋 Project Modules

1. **Data Importing** - Open CSV and convert to XLSX format
2. **Data Cleaning** - Remove empty rows, duplicates, fix spelling and whitespace
3. **Data Formatting** - Adjust column widths, fix split department names using Flash Fill
4. **Pivot Table Analysis** - Summarize equipment count by department and equipment class
5. **Insight Extraction** - Identify top departments and dominant equipment types

## 📈 Key Results

| Analysis | Finding |
|---|---|
| Top department by equipment | Transportation — 1,221 units (77%) |
| Total fleet size | 1,582 units across 12 departments |
| Most common equipment class | Transit Bus — 379 units |
| Departments with under 10 units | 3 departments |
| Cleaned data issues fixed | Duplicates, blanks, spelling, whitespace, split columns |
